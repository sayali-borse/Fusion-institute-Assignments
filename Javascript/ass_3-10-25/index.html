JAVASCRIPT

  • JavaScript (JS) is a high-level, interpreted programming language.
  • It was created in 1995 by Brendan Eich.
  • Runs directly in the browser to make web pages dynamic and interactive.
  • Together with HTML (structure) and CSS (styling), JavaScript is the third core technology of the web.

Features of JavaScript

  1. Lightweight & Easy to Learn
  2. Cross-platform (runs on all browsers & OS)
  3. Event-driven (reacts to clicks, inputs, etc.)
  4. Object-Oriented (can create objects, use prototypes)

Where is JavaScript Used?

  1. Web Development (Frontend): Forms, animations, validations.
  2. Backend Development: Server-side apps with Node.js.
  3. Mobile Apps: React Native, Ionic.
  4. Game Development: Phaser.js, Babylon.js.
